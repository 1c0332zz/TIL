## 2. 목록 (list)

### 순서가 없는 리스트 : -, *

    - 계란
      - 밥
  - 비빔면
- 사과

## 코드 블록

* `(backtick) 기호 3개를 활용하여 작성한다. (```)

- 특정 언어를 명시하면 syntax Highlighting적용 가능

  - 일부 환경에서는 적용이 되지 않을 수 있음

  ```python
  print("hello")
  # 주석
  ```
